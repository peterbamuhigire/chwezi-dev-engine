# Recommended Books and Reading Material

## Must Encode Now

| Title | Author/issuer | Gap filled |
| --- | --- | --- |
| Accelerate | Nicole Forsgren, Jez Humble, Gene Kim | Evidence-based software delivery and DevOps performance metrics. |
| Designing Data-Intensive Applications | Martin Kleppmann | System design depth for data, distributed systems, and reliability. |
| Building Microservices, 2nd ed. | Sam Newman | Service boundaries, deployment, and operability patterns. |
| Security Engineering, 3rd ed. | Ross Anderson | Security thinking beyond checklist-level guidance. |
| Site Reliability Engineering | Betsy Beyer et al. | Operational reliability and incident-management discipline. |
