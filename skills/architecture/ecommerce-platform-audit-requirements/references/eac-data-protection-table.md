|---|---|---|

| Burundi | Loi no. 1/03 du 10 mars 2026 | Authority being operationalised; treat as in force and seek local review. |
| DRC | Ordonnance-Loi 23/010 of 13 March 2023, Code du numerique | Digital Code provisions in force; no standalone DPA in this table. |
| South Sudan | No comprehensive data-protection law confirmed in the source report | Use strict programme default and seek local legal review. |

## Use Rules

- Treat this as compliance context, not legal advice.
- Where multiple countries are involved, default to the strictest applicable programme control.
- Always capture consent and minimise data for donor programmes.
- For public reports, anonymise or aggregate confidential company data.
- Escalate cross-border transfer, children's data, financial data, and sensitive personal data questions.
