# Control Room Registers

## Workstream Tracker

| Field | Description |
|---|---|
| Workstream | Bid compliance, methodology, CVs, pricing, outreach, diagnostics, systems, reporting, etc. |
| Accountable owner | Exactly one person. |
| Contributors | Supporting roles. |
| Start/due date | Planned dates. |
| Status | Not started, in progress, blocked, review, done. |
| Dependency | Upstream/downstream dependency. |
| Next action | Concrete next step. |

## Decision Log

- Decision ID.
- Date.
- Decision required.
- Options considered.
- Decision made.
- Rationale.
- Owner.
- Consequences.
- Reversal trigger.

## Evidence Register

- Evidence ID.
- Related claim/deliverable.
- Source file or URL.
- Source tier.
- Date verified.
- Owner.
- Confidentiality level.
- Reuse permission.
