# Asset acceptance and mobile budgets

Back to [Game 3D Asset Pipeline](../SKILL.md).

Accept only when scale, pivot, normals, topology, UVs, bake, material channels, rig, skin, clips, collider, LOD, import, warnings, memory, load and stress-scene results pass. Retain screenshots and profiler capture.

Budget triangles/vertices, materials, compressed texture memory, bones/influences, colliders, shadows, lights, transparency/overdraw, particles, shader cost, animation memory and installed contribution by asset class and simultaneous count. Derive numbers from target-device scenes. The books' 30,000-triangle character and 4096 texture examples are tutorial context, not mobile budgets.
