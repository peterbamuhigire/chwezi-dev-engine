# Studio operating gates

A greenlight packet states player/product hypothesis, observed evidence, technical spikes, platform/compliance risks, content throughput, staffing gaps, cost/runway range, commercial path, stop conditions, and accountable approvers. A milestone cannot pass on percentage-complete reporting alone.

Estimates expose scope units, historical/benchmarked basis, assumptions, optimistic/likely/pessimistic range, integration and rework allowance, external dependencies, confidence, and re-estimation trigger. Vendor acceptance covers rights/provenance, editable source, naming/version, technical budgets, security transfer, defect remedy, and integration proof.
