# Red-Team Scorecard

## Evaluator Simulation Steps

1. Read the evaluation grid without the draft open.
2. Restate what the evaluator is likely looking for under each criterion.
3. Score the draft strictly against visible evidence only.
4. Penalise unsupported claims, unclear ownership, generic methodology, and missing proof.
5. Calculate weighted score and threshold verdict.
6. List fixes by score impact, not by writing preference.

## Scorecard Fields

| Field | Description |
|---|---|
| Criterion | Buyer criterion. |
| Weight/points | Official weight. |
| Evidence in draft | Where the answer appears. |
| Assessor concern | Why an evaluator may withhold points. |
| Predicted score | Conservative score. |
| Fix required | Concrete fix. |
| Owner | One accountable owner. |
| Gate status | Pass, fix, blocked. |
